About Me

Nama saya Muhammad Ilmi Fasya Ar Rafi. Saya lahir di Bandung pada 20 November 2006. Saat berusia sekitar tiga sampai empat bulan, saya pindah ke Kota Malang. Saya tinggal di sana hingga usia tiga tahun. Setelah itu, saya berpindah ke Pulau Bali dan menetap sampai usia lima tahun. Menjelang usia enam tahun, saya kembali ke Bandung. Sejak saat itu hingga sekarang, Bandung menjadi tempat saya tumbuh, belajar, dan membentuk arah hidup. Perpindahan kota sejak kecil membiasakan saya beradaptasi. Saya cepat menyesuaikan diri dengan lingkungan baru. Saya terbiasa melihat perbedaan sebagai hal wajar. Pengalaman ini membentuk cara saya berpikir terbuka dan fleksibel.

Saya menempuh pendidikan dasar di SDN 127 Sekeloa. Di sekolah ini saya membangun kebiasaan belajar dan disiplin sejak dini. Saya melanjutkan ke SMPS Pasundan 3 Bandung. Di jenjang ini saya mulai aktif berdiskusi dan berani menyampaikan pendapat. Pendidikan menengah atas saya jalani di SMAS Pasundan 2 Bandung. Di sini saya semakin terbiasa berpikir kritis dan terstruktur. Saya belajar menyusun argumen dengan jelas. Modal penting untuk dunia kuliah dan kerja
